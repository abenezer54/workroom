# Workroom

Workroom is a client portal SaaS for agencies, freelancers, and consultants.

It helps service businesses manage clients, projects, tasks, invoices, and project updates in one professional workspace.

## Project Status

Planning phase.

## Tech Stack

Frontend:
- Next.js
- TypeScript
- Tailwind CSS

Backend:
- Go
- Gin
- PostgreSQL
- GORM
